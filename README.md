#### Criando um projeto React + Bootstrap

* npm create vite@latest nomeDoProjeto
* selecionar o react
* tipo javascript ou typescript 

* comando para rodar o projeto
npm run dev

#### Informações
Muito usando em 

* Angular, NextJS, ReactJS
* NestJS, PrismaJS, FastiFy

#### Json server

npm install json-server

banco.json
